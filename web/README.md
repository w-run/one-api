# mimi-router 的前端界面

> 本仓库使用单一主题：default（基于 Berry 主题）

## 主题：default

由 [MartialBE](https://github.com/MartialBE) 开发的 Berry 主题，经 [W/Run](https://github.com/w-run) 修改后作为默认主题。

特性：
- 暗色模式参考 VSCode Dark++ 配色
- 简洁现代的 UI 设计
- 响应式布局

## 构建说明

1. 修改 `web/THEMES` 文件，当前内容为 `default`
2. 构建命令：`npm run build && mv -f build ../build/default`
